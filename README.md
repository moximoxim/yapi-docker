# yapi-docker

- 环境：
  - 硬件要求：`1 GB RAM minimum`
  - 工作路径：`mkdir -p /opt/git-yapidata`
  - ip:xxx.xxx.xxx.xxx
- 克隆：
  - `cd /opt/git-yapidata && git clone https://github.com/xingyalee/yapi-docker.git`
- 修改权限：
  - `chmod -R 777 /opt/git-yapidata`
- 运行指令：
  - `docker-compose up -d`
- chrome浏览网页：
  - `http://ip:3000`
