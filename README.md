# yapi-docker

- env：
  - Hardware requirements：`1 GB RAM minimum`
  - work path：`mkdir -p /opt/git-yapidata`
  - ip:xxx.xxx.xxx.xxx
- clone：
  - `cd /opt/git-data && git clone https://github.com//yapi-docker.git`
- permission：
  - `chmod -R 777 /opt/git-yapidata`
- run command：
  - `docker-compose up -d`
- open chrome：
  - `http://ip:3000`
