# yapi-docker

- 环境：
  - 工作路径：`mkdir /docker-yapi`
  - ip:xxx.xxx.xxx.xxx
- 克隆：
  - `cd /docker-yapi && git clone https://github.com/xingyalee/yapi-docker.git`
- 修改权限：
  - `chmod -R 777 /docker-yapi`
- 运行指令：
  - `docker-compose up -d`
- chrome浏览网页：
  - `http://ip:3000`
